Prototype Pattern — Step-Through

Document — doc1
title: "Report Q1"
content: "Revenue: $1M"
styles: {font: Arial, color: blue}
Document — doc2 (copy)
title: "Report Q1"
content: "Revenue: $1M"
styles: {font: Arial, color: blue}
✗ Shallow copy — shared styles
doc2.styles["color"] = "red"
doc1.styles → {font: Arial, color: red}
Same dict object in memory
✓ Deep copy — independent styles
doc2.styles["color"] = "red"
doc1.styles → {font: Arial, color: blue}
copy.deepcopy() duplicates nested objects
